# Tiny TV

This is a little Vue app that powers a small retro-style TV on my desk.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
 
### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
